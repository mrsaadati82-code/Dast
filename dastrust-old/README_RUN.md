# دست راست — نسخه اپلیکیشن واقعی (MVP عملیاتی)

این نسخه دیگر فقط پیش‌نمایش نیست و شامل بک‌اند محلی، ورود/ثبت‌نام، پنل ادمین، ذخیره‌سازی داده‌ها و اتصال قابل تنظیم به AI است.

## اجرا

```bash
npm install
npm start
```

سپس مرورگر را باز کنید:

```text
http://localhost:8787
```

## حساب ادمین پیش‌فرض

در اولین اجرا اگر دیتابیس وجود نداشته باشد، حساب زیر ساخته می‌شود:

```text
Email: admin@dastrast.local
Password: Admin12345
```

## امکانات پیاده‌سازی‌شده

- ورود و ثبت‌نام کاربر
- نقش ادمین و کاربر عادی
- پنل ادمین
- تنظیم ارائه‌دهنده هوش مصنوعی: Local، OpenAI، OpenRouter، Groq، Custom
- ذخیره توکن AI در بک‌اند، نه در کلاینت
- تست اتصال AI از پنل ادمین
- ثبت تراکنش دستی
- ثبت تراکنش از متن فارسی و اصطلاحات روزمره
- ثبت تراکنش از چت دستیار
- ورودی صوتی در مرورگرهای پشتیبان Web Speech API
- ارسال عکس رسید و تحلیل از طریق مدل Vision در صورت اتصال AI
- گزارش‌های تحلیلی و پرسش از داده‌ها
- مدیریت چک و تعهدات مالی
- شبیه‌ساز و API تحلیل پیامک بانکی
- تاریخ و اعداد فارسی / شمسی
- تم روشن و تاریک با رنگ‌بندی برند

## نکته مهم درباره SMS و نوتیفیکیشن واقعی

مرورگر وب به دلایل امنیتی اجازه خواندن مستقیم SMS گوشی را نمی‌دهد. برای دریافت واقعی SMS باید یکی از این مسیرها اضافه شود:

1. اپ Android Native / React Native با مجوز SMS
2. اتصال به سرویس پیامکی یا بانکداری باز
3. PWA Push Notification با Service Worker و سرور Push

در این نسخه API و UI آماده است و بخش پیامک به صورت شبیه‌ساز/ورودی دستی کار می‌کند.

## محل ذخیره داده‌ها

داده‌ها در فایل زیر ذخیره می‌شوند:

```text
data/db.json
```

برای ریست کامل، این فایل را حذف کنید و سرور را دوباره اجرا کنید.
