دست راست — فاز ۱۰ (یکپارچگی کامل دفتر + UX موبایل + گزارش‌های پیشرفته)
======================================================================

اجرا: npm install && npm start → http://localhost:8787
ادمین: admin@dastrast.local / Admin12345
⚠️ برای داده‌های قدیمی، دفتر به‌صورت خودکار یک‌بار بازسازی می‌شود
   (ledgerVersion). نیازی به حذف دستی db.json نیست.

== ۱) منوی لمسی/کلیک‌راست روی همهٔ لیست‌ها ==
کامپوننت ActionRow روی این لیست‌ها فعال شد: تراکنش‌ها (تاریخچه + دفتر
حساب شخص)، چک‌ها، حساب‌ها، دسته‌بندی‌ها، پروژه‌ها، فاکتورها. کلیک‌راست
(دسکتاپ) و لمس طولانی (موبایل) → منوی داخلی عملیات.

== ۲) رفع باگ بحرانی: تراکنش حذف‌شده در گزارش‌ها ==
علت: صورت سود و زیان از «تراکنش‌ها» می‌خواند ولی تراز از «دفتر اسناد».
حالا همهٔ گزارش‌ها (تراز، سود و زیان، جریان نقد، مانده حساب‌ها) فقط از
«دفتر اسناد» (تک‌منبع حقیقت) محاسبه می‌شوند. با حذف هر تراکنش، سند
مرتبط حذف و همهٔ گزارش‌ها بلافاصله هماهنگ می‌شوند.
+ بازسازی خودکار دفتر برای دیتابیس‌های قدیمی ناهماهنگ (یک‌بار).

== ۳) نمایش بدهکار/طلبکار ==
مانده‌ها در فهرست اشخاص، مشتریان و مطالبات درست محاسبه و نمایش داده
می‌شوند (تب طلبکار/بدهکار + گزارش سن مطالبات).

== ۴) رفع باگ چیدمان موبایل ==
کانتینر اصلی به h-[100dvh] با flex تبدیل شد؛ حالا در صفحاتی که محتوا کم
است هم منوی پایین و بالا سرجای خود می‌مانند و صفحه جمع نمی‌شود.

== ۵) تکمیل گزارش‌ها ==
- صورت سود و زیان: محاسبه از دفتر (عملیاتی/غیرعملیاتی، بهای تمام‌شده،
  سود ناخالص، حاشیه سود، بازه، نمودار).
- جریان نقدی: تفکیک عملیاتی/سرمایه‌گذاری/تامین مالی + بازه +
  «پیش‌بینی جریان نقد آینده» بر اساس چک‌های پاس‌نشده.
- چک‌ها: گزارش چک‌های در جریان وصول (cheques/report) + هشدارها.

== منطق حسابداری ==
تک‌منبع حقیقت = سند حسابداری دوطرفه. فروش/خرید نسیه به حساب‌های
دریافتنی/پرداختنی می‌رود (نه نقد). هر عملیات یک سند متوازن می‌سازد و
ویرایش/حذف، سند را بازمی‌سازد. همه‌چیز بین بخش‌ها سینک است.

تست یکپارچگی (هر دو سرور): فروش۱۰+طلب۵+هزینه۳+واریز۲+انتقال۱ →
صندوق۶، بانک۳، علی+۵، تراز متوازن، سود۱۲، جریان‌نقد درست،
حذف تراکنش → سود/تراز فوراً صفر، پیش‌بینی نقد با چک‌ها درست.

UI/UX: پاپ‌آپ‌های داخلی، تقویم شمسی، رنگ‌های با کنتراست درست (glass-btn).
