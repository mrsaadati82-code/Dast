دست راست — فاز ۴ (تکمیل کامل داشبورد + دستیار)
================================================

اجرا:
  npm install
  npm start          → http://localhost:8787
یا ویندوز بدون نصب: RUN-WINDOWS-NO-INSTALL.bat
ادمین پیش‌فرض: admin@dastrast.local / Admin12345

== داشبورد مالی (کامل شد) ==
- کتابخانه نمودار زیبا و متحرک (SVG درون‌خطی، بدون وابستگی خارجی):
  • میله‌ای گروهی درآمد/هزینه ۱۲ ماه
  • خطی: سود خالص ماهانه
  • سطحی: مانده تجمعی
  • دونات: سهم هزینه‌ها / سهم درآمدها / روش ثبت
  • میله افقی: اشخاص برتر / پروژه‌ها / مانده حساب‌ها
  • گِیج: نرخ پس‌انداز از درآمد
- کاربر خودش انتخاب می‌کند چه نمودارها و چه KPIهایی نمایش داده شوند:
  دکمه چرخ‌دنده بالای داشبورد → پنل «شخصی‌سازی داشبورد».
  انتخاب‌ها روی سرور ذخیره می‌شوند و ماندگارند.
- بازه زمانی: هفته/ماه/فصل/سال/کل
- فیلتر پیشرفته: دسته‌بندی، شخص، پروژه، بانک
- مقایسه دوره‌ای با درصد تغییر
- هشدارهای مالی هوشمند
Endpointهای جدید:
  GET /api/analytics/summary?range=&category=&personId=&projectId=&bank=&type=
  GET /api/analytics/filters
  GET /api/analytics/compare
  GET /api/analytics/alerts
  GET/PUT /api/dashboard/prefs

== دستیار هوشمند (کامل شد) ==
- حذف با تایید دو مرحله‌ای: «آخرین تراکنش رو حذف کن» → دکمه بله/انصراف.
- بازیابی (Undo/Restore): دکمه «بازگردان» زیر پیام؛ هم ثبت‌ها و هم حذف‌ها.
- ویرایش چک با دستور طبیعی: «مبلغ آخرین چک رو ... کن / سررسید ... / پاس شد».
- یادگیری از اصلاح کاربر: «دنگ شام» → دستهٔ «رستوران و کافه»، دفعهٔ بعد خودکار.
- دکمه «اصلاح دسته‌بندی» روی پیام برای آموزش سریع.
- (از فاز ۳) رفع ابهام با دکمه شیشه‌ای، چند عملیات بدون جداکننده، تشخیص نام دقیق.
Endpointهای جدید:
  POST /api/assistant/confirm-delete   { kind, id }
  POST /api/assistant/undo
  POST /api/assistant/correction       { text, field, value }
